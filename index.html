Pokémon TCG Price Trackerclose

Our first big project was a price tracker that searches Pokémon trading cards and keeps track of their price.

My role was the front-end lead but I helped write the back-end for the individual card page and other places.

It features a favorites list using local storage, and clicking on a card's art after you search it brings up an individual card page that can be shared.

There is also a cryptocurrency conversion that converts the price of the card in USD to a selection of popular cryptocurrencies.

Weather Dashboardclose

This assignment was to make a simple weather dashboard that uses API calls to get weather data to append to the page.

My dashboard is mobile/tablet optimized and the weather data is sourced using calls to OpenWeatherAPI.

It uses local storage to store the text input last searched and a clear button to empty the search list if it gets too long.
