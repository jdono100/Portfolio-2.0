Our first big project was a price tracker that searches Pokémon trading cards and keeps track of their price.

My role was the front-end lead but I helped write the back-end for the individual card page and other places.

Day Schedulerclose

This project is a single page day planner that uses local storage to save information into labelled timeblocks.

It uses Moment.js to display the date/time and Bootstrap for the front-end framework.

The timeblock backgrounds change color depending on if they're in the past (grey), the current hour (red), or the future(green). All work was done by me.
